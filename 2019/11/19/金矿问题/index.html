金矿问题

2019-11-19 小灰の算法之旅 评论

动态规划

  • 把复杂的问题简化成规模较小的子问题，再从简单的子问题自底向上一步一步递推，得到复杂问题的最优解

  • 最优子结构

  • 状态转移方程式

    • 金矿数量为n，工人数量为w，含金量为数组g[]，开采人数为数组p[]

      • F（n,w）= 0 （n=0或w=0）

      • F (n,w) = F（n-1,w）(n大于等于1，w<p[n-1])

      • F (n,w) = max(F(n-1,w),F(n-1,w-p[n-1])+g[n-1])

        (n大于等于1,w大于等于p[n-1])

    • 递归

      • 时复杂 = O(2^n)

    • 自底向上 二维数组

      • 时 空 复杂 O(nw)

    • 一维数组

      • 空 复杂 O(n)
